\section{How to run}

Follow the steps below to set up the project in your local environment:

\begin{enumerate}

\item Clone the repository:

\begin{minted}[frame=lines,framesep=2mm,baselinestretch=1.2,bgcolor=gray!7!white, fontsize=\footnotesize,numbersep=2mm]{bash}
    git clone https://github.com/arthurabello/trabalho_lp_A2.git
    cd trabalho_lp_A2
\end{minted}

\item Create and activate a virtual environment (recommended):

\begin{minted}[frame=lines,framesep=2mm,baselinestretch=1.2,bgcolor=gray!7!white, fontsize=\footnotesize,numbersep=2mm]{bash}
    python3 -m venv venv
    source venv/bin/activate
\end{minted}
 
\item Install the dependencies:

\begin{minted}[frame=lines,framesep=2mm,baselinestretch=1.2,bgcolor=gray!7!white, fontsize=\footnotesize,numbersep=2mm]{bash}
    pip install -r requirements.txt
\end{minted}

\item Run the main script to play Warbound:

\begin{minted}[frame=lines,framesep=2mm,baselinestretch=1.2,bgcolor=gray!7!white, fontsize=\footnotesize,numbersep=2mm]{bash}
    cd src
    python3 main.py
\end{minted}

\end{enumerate}
